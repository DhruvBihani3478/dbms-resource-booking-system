# 📚 College Resource Booking System (CRBS)

A **Database Management System (DBMS)** mini project developed to simplify the booking and management of college resources such as laboratories, classrooms, seminar halls, and equipment.

The project demonstrates the implementation of **Oracle SQL/PLSQL** concepts along with a clean and responsive frontend built using **HTML, CSS, and JavaScript**.
