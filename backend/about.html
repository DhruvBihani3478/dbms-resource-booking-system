About CRBS

College Resource Booking System

The College Resource Booking System (CRBS) is a web-based application developed to simplify the reservation of college resources such as laboratories, classrooms, seminar halls, and equipment.

This project is built using HTML, CSS, JavaScript and Oracle PL/SQL. It demonstrates database design, SQL queries, PL/SQL programming, procedures, functions, triggers, views, indexing and transaction management.


Project Features

  • User Management
  • Resource Management
  • Resource Booking
  • Booking Cancellation
  • Reports & Analytics
